# ✨ 2025-KuTTy 조직 소개 (시간표 추천 서비스)

## 👋 KuTTy 프로젝트 개요

**프로젝트명:** KuTTy (시간표 추천 서비스: Konkuk UNIV. Time Table Yield)

**핵심 기술:** SW

**수행 기간:** 2025.04.01 ~ 2025.10.31

### 프로젝트 소개 및 목표

기존 시간표 서비스의 한계를 극복하고, 사용자의 세부 조건(공강 요일, 시간대, 전공/교양 등)을 충분히 반영하는 **사용자 맞춤형 시간표 추천 시스템** 개발을 목표로 합니다.

### 주요 기능

* **텍스트/태그 기반 맞춤 시간표 추천 알고리즘** 구현
* 사용자 데이터 기반 **추천 알고리즘 제공** (다른 유사 사용자 패턴 반영)
* **졸업 요건 확인 및 시각화** 기능 지원
